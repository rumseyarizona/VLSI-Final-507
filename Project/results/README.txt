PicoRV32 Branch Predictor + Sky130 SRAM Integration Results

This folder contains the final local SiliconCompiler/OpenROAD build artifacts used for the project report.

source_snapshot/
  RTL, run script, SRAM setup script, and SDC constraints.

sram/
  SRAM LEF, Liberty timing file, and compressed detailed GDS.

final_gds/
  Final GDSII output from write.gds.

final_views/
  Final Verilog/DEF/LEF/view files generated by write.views when available.

metrics/
  Final metrics JSON used for report tables.

logs/
  Key SiliconCompiler/OpenROAD logs from synthesis, floorplan, placement, CTS, routing, and GDS export.

figures/
  Auto-generated PNGs from OpenROAD reports.

Notes:
- The run was performed locally using SiliconCompiler/OpenROAD.
- OpenROAD version: 26Q1-951-g6975124cf2.
- Yosys version: 0.64.
- KLayout version: 0.28.16.
- place.repair_design and cts.repair_timing were bypassed in the final layout run due to memory limitations.
- The final GDS demonstrates SRAM macro integration and routing but should not be claimed as full signoff timing-clean.
